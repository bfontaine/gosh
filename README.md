# Gosh!

**Gosh** is a shell written in Go.

## Install

    go get github.com/bfontaine/gosh

### Dependencies

* Go 1.2 or higher
* Readline

## Features

Just a basic REPL, right now.

### Builtin commands

* `cd <path>`: change the current directory (can contain spaces)
* `quit`, `exit`, `^D`: exit the shell
